RECV
====

Parameter Considerations
------------------------

**NBYTE**
:   TCP/IP for MVS 3.8 Assembler limits the NBYTE value to
    1024K (1048576) bytes.

Unsupported Parameters
----------------------

The following keyword parameters are not supported by TCP/IP for MVS 3.8
Assembler:

-   ALET
-   ECB
-   ERROR
-   FLAGS
-   REQAREA
